# 使用说明（面向业务/产品同学）

## 你现在只需要 1 条命令

```bash
./scripts/dev.sh web
```

它会自动准备项目内局部环境（`.venv`），并启动网页聊天界面。

## 本版本重点

- 所有运行时依赖都在项目内隔离（`.venv`）
- 模型缓存也在项目目录（`.cache/huggingface`）
- 启动流程简化为单脚本
- 保留混合检索（BM25+向量）+ Rerank + 自动去重

## 运行模式

```bash
./scripts/dev.sh ingest  # 增量入库
./scripts/dev.sh chat    # 命令行问答
./scripts/dev.sh web     # 网页问答
```

## 首次配置

脚本首次运行会自动生成 `.env`（来自 `.env.example`），你只要补充：

- `DEEPSEEK_API_KEY`
- `QDRANT_URL`
- `QDRANT_API_KEY`

## 查询链路

1. 向量召回（Qdrant）
2. BM25 召回（本地缓存索引）
3. 分数融合
4. 文本去重
5. Rerank 重排
6. DeepSeek 生成
